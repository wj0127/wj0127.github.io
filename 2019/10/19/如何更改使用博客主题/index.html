如何更改使用博客主题

2019-10-19
本文总阅读量次

首先在浏览器https://hexo.io/themes/进入github官方主题界面，如图

在里面选择一个自己喜欢的主题，点进去就像我的这一个

点击ANTIQUITY就会进入这个主题拥有者的博客里

进入之后复制主题的地址

复制好之后下载![img](file:///C:\Users\86158\Documents\Tencent Files\1265267243\Image\C2C\H`PPET%YSFW]}S_KCC16ZD7.png)

下载完之后会在themes 目录下生成一个名为antiquity文件

然后走一下命令

cd themes/antiquity

git pull //获取远程内容

更改一下hexo工程目录下的配置文件_config.yml，主题名修改一下即可 theme: antiquity

切换到blog目录

hexo g # 重新生成静态文件

hexo s # 启动本地web服务器

现在打开http://localhost:4000，会看到我们已经应用了一个新的主题。

然后上传到，远程仓库就行了

完
